## IntelliJ
1. To Run code, press  ⌃   R  or click the   icon in the gutter.
2. Press  ⌥   ↩ Return  with your caret at the highlighted text to see how IntelliJ IDEA suggests fixing it.
3. Press  ⌃   D  to start debugging your code. We have set one   breakpoint for you, but you can always add more by pressing  ⌘   F8 .
